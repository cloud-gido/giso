# iOS 长视频 Demo

与 `examples/android-video-demo` 对称，验证 `video-ios-beta` App Key + longvideo 空间校验。

## 运行

```bash
cd examples/ios-video-demo
export GISO_ENDPOINT=http://localhost:8123/v1/track
swift run
```

或在 Xcode 中 Open Package，选择 iOS Simulator 运行。

| Key | 环境 |
